# CALC.PY
🧮 Calculator App using Python Tkinter

A modern desktop calculator built with Python and Tkinter, featuring a clean dark-themed user interface similar to a mobile calculator.

✨ Features

- Basic arithmetic operations:
  - Addition ("+")
  - Subtraction ("-")
  - Multiplication ("×")
  - Division ("÷")
- Percentage ("%") calculation
- Positive/Negative ("+/-") toggle
- Real-time result preview
- Keyboard support
- Error handling for invalid expressions and division by zero
- Dark mode user interface

🛠️ Requirements

- Python 3.x
- Tkinter (comes pre-installed with most Python installations)

🚀 Installation

1. Clone the repository:

git clone https://github.com/your-username/tkinter-calculator.git

2. Navigate to the project directory:

cd tkinter-calculator

3. Run the application:

python calculator.py
